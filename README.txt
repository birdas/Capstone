----SET UP----

Step 1: Make sure you have anaconda installed for python environments (or manually install the packages needed on your desired environment)

Step 2: Run "conda env create -f environment.yml"

Step 3: Run "conda activate Definitions" or "source activate Definitions"

Step 4: Make sure your default/fallback microphone is set to the desired input. If you are wanting to record desktop audio then set your microphone to the monitor of your output device.

----RUNNING THE PROGRAM----

Step 1: Run voice.py
